## What is it ?

KRDragImageView can drag and move an image on the screen.

## How To Get Started

``` objective-c
#import "KRDragImageView.h"

-(void)viewDidLoad
{
    [super viewDidLoad];
    KRDragImageView *_krDragImageView = [[KRDragImageView alloc] initWithImageName:@"green.png"];
    [self.view addSubview:_krDragImageView];
}
```
